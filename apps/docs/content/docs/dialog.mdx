### Installation

<Tabs items={["npm", "pnpm", "yarn", "bun"]}>
  <Tab value="npm">```shell npm install @realpay-ui/dialog ```</Tab>
  <Tab value="pnpm">```shell pnpm install @realpay-ui/dialog ```</Tab>
  <Tab value="yarn">```shell yarn add @realpay-ui/dialog ```</Tab>
  <Tab value="bun">```shell bun add @realpay-ui/dialog ```</Tab>
</Tabs>

### Usage

```typescript
import {
  Dialog,
  DialogContent,
  DialogDescription,
  DialogHeader,
  DialogTitle,
  DialogTrigger,
} from "@realpay-ui/dialog";

// or

import * as BaseDialog from "@realpay-ui/dialog";
```

```typescript
<Dialog>
  <DialogTrigger>Open</DialogTrigger>
  <DialogContent>
    <DialogHeader>
      <DialogTitle>Are you absolutely sure?</DialogTitle>
      <DialogDescription>
        This action cannot be undone. This will permanently delete your account
        and remove your data from our servers.
      </DialogDescription>
    </DialogHeader>
  </DialogContent>
</Dialog>
```
