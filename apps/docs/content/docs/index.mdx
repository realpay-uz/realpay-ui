```

```

### Link

You can use the buttonVariants helper to create a link that looks like a button.

<Tabs items={["Preview","Code"]}>
<Tab value="Code">
```tsx
import {Button} from "@realpay-ui/button";

export const LinkThatLooksLikeAButton = () => {
    return <Button variant="link">
   This a link (button)
    </Button>
}
```
</Tab>
<Tab value="Preview">
<Button variant="link">
   This a link (button)
    </Button>
</Tab>
</Tabs>

Alternatively, you can set the asChild parameter and nest the link component.

<Tabs items={["Preview", "Code"]}>
  <Tab value="Code">
    ```tsx
    <Button asChild>
      <Link href="/login">Login</Link>
    </Button>
    ```
  </Tab>
  <Tab value="Preview">
    <Button asChild>
      <Link href="/login">Login</Link>
    </Button>
  </Tab>
</Tabs>
